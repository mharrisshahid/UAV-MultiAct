Abstract

Human action recognition is a core requirement for intelligent surveillance, yet fixed ground-based cameras suffer from limited viewpoints, occlusions, and poor adaptability to large-scale outdoor environments, while aerial UAV-based recognition remains challenging due to small target sizes, oblique views, and complex multi-person interactions. To address these challenges using practical RGB-only sensing, we propose a unified multi-cue framework for multi-person action recognition in UAV surveillance videos that integrates silhouette extraction, ridge and distance transform-based structural cues, SuperPoint–LightGlue keypoint matching for robust correspondence, pose estimation for articulated motion modeling, and tracking-any-point for temporal consistency. The heterogeneous features are fused into a joint representation and modeled using a transformer-based classifier to capture long-range temporal dependencies and interaction dynamics. Experiments conducted on two public UAV datasets demonstrate the effectiveness of the proposed approach, achieving 88.90% accuracy on MOD20 and 88.78% on CHAD, outperforming several recent methods and confirming the robustness of the framework for aerial multi-person action understanding.

Method

The proposed framework provides an end-to-end pipeline, as shown in Fig., for human action recognition in challenging UAV-based multi-person scenarios by utilizing complementary visual and motion cues. The framework begins with preprocessing steps, including the redundant frames removal, spatial resizing, and intensity normalization. From each preprocessed image, human silhouettes are extracted, followed by ridge detection and distance transform. Concurrently, visual features are learned using SuperPoint and LightGlue for precise feature matching under viewpoint transformations. Human pose estimation is employed to encode body motion, and a Tracking Any Point system is used for temporal keypoint alignment. Finally, all the extracted information is aggregated into a unified feature and passed to a transformer to encode multi-person action recognition.

Datasets

  • MOD20 Dataset
  • CHAD Dataset

This study utilized the MOD20 dataset, containing 2,324 video clips across 20 action classes. We also used the CHAD dataset which includes 1.15 million frames across 22 anomalous behaviors and some normal classes.
